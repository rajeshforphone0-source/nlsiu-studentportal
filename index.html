  1. 1Go to Google Cloud Console → New Project
  2. 2Enable the Google Calendar API
  3. 3Credentials → Create → OAuth 2.0 Client ID, type: Web application
  4. 4Add your page URL as Authorised JS Origin
  5. 5Paste Client ID in the hero section above
